ALGORITMO DEL FILTRO DE PARTÍCULAS:
───────────────────────────────────

Inicialización: 1000 partículas distribuidas uniformemente

Para cada ciclo (cada /odometry + /scan sincronizados):

  ┌─ PREDICCIÓN: x_k^{(i)} ~ p(x_k | u_k, x_{k-1}^{(i)})
  │  x += dt * v * cos(θ)
  │  y += dt * v * sin(θ)
  │  θ += dt * ω
  │  + ruido Gaussiano N(0, sigma)
  │
  ├─ ACTUALIZACIÓN: w_k^{(i)} = p(z_k | x_k^{(i)}) * w_{k-1}^{(i)}
  │  Para cada scan:
  │    Simular scan desde partícula i
  │    Comparar con scan real
  │    w_i *= exp(-distancia²/sigma²)
  │
  ├─ NORMALIZACIÓN: w_i = w_i / Σ(w)
  │
  ├─ REMUESTREO: Si Neff < threshold
  │  Muestrear con reemplazo según pesos
  │  Igualar todos los pesos a 1/N
  │
  └─ ESTIMACIÓN:
     x_est = Σ(w_i * x_i)
     y_est = Σ(w_i * y_i)
     θ_est = atan2(Σ(w_i*sin(θ_i)), Σ(w_i*cos(θ_i)))
     localized = Var(particulas) < threshold
